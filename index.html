Wildberries product position

This work is a program written for parsing product positions in the search on the Wildberries platform. It allows you to track the position of products in the search results for a given article and key query, as well as output all the data received in Excel format. This project can be useful for monitoring the position of your products and competitors on the Wildberries platform.

Данная работа представляет собой программу, написанную для разбора позиций товаров в поиске на платформе Wildberries. Она позволяет отслеживать позиции продуктов в результатах поиска по заданной статье и ключевому запросу, а также выводить все полученные данные в формате Excel. Данный проект может быть полезен для мониторинга позиций ваших продуктов и конкурентов на платформе Wildberries.

Yula parsing

This program parses products from the Yula website, saves links to products and parses detailed data about products from these links, saving the results to an Excel file. The program is useful for analyzing prices and creating a database of goods. The project can be used for the portfolio of a programmer who is engaged in task automation.

Эта программа анализирует товары с сайта Юла, сохраняет ссылки на товары и анализирует подробные данные о товарах по этим ссылкам, сохраняя результаты в файл Excel. Программа полезна для анализа цен и создания базы данных товаров. Проект может быть использован для портфолио программиста, занимающегося автоматизацией задач.
